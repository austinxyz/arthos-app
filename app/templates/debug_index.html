API Endpoints

🔧 Trigger Scheduler

Manually trigger the scheduler to fetch stock data. Can bypass market hours check for testing.

POST /debug/scheduler-log/trigger?bypass_market_hours=true
🔧 Trigger RR History Update

Manually trigger the RR history update to recalculate net costs. Can bypass market hours check for testing.

POST /debug/rr-history-log/trigger?bypass_market_hours=true
🔧 Trigger Option Data Fetch

Manually trigger the option data fetch job to update cached option strategies for all watchlist tickers.

POST /debug/options-cache-log/trigger
